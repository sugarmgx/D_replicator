1. D_Replicator とは

D_Replicator は、1つのオブジェクトを何百〜何千個に複製し、その並びと動きをリアルタイムにプレビューしながら作れる Blender アドオンです。

  • 非破壊 — 元のオブジェクトはそのまま。設定を変えれば即座に反映されます。
  • 軽量 — 実体を大量に作らず「表示専用」で描画するので、数千個でも軽快です。
  • 直感的 — 追加した瞬間から扱いやすい初期状態。自然に作り込めます。

モーショングラフィックス、配列・群衆、装飾、背景の作り込み、ループアニメーションなどに向いています。

NOTE本マニュアルは 第1部 クイックスタート(まず使えるように)と 第2部 リファレンス(全機能の詳細)の2部構成です。
QUICK START

2. インストール

Blender 5.1 推奨(4.2 以降で動作)。Windows / macOS 対応。レンダラーは EEVEE / Cycles。

  • 編集 ▸ プリファレンス ▸ 拡張機能(Get Extensions) を開く
  • 右上の ▼ ▸ ディスクからインストール… を選ぶ
  • 配布された D_Replicator-x.x.x.zip を選択する
  • 一覧の D_Replicator にチェックを入れて有効化する
  • 3D ビューポート ▸ サイドバー(N キー) ▸ Replicator タブが表示されます
CAUTION有効化できないとき: いったん Blender を再起動してください。それでも直らない場合は、セキュリティソフトで Blender の拡張機能フォルダを除外設定にしてから入れ直すと確実です。

3. はじめての複製 — 3ステップ

  • ① 複製したいオブジェクトを選択(例: 立方体)
  • ② N パネル ▸ Replicator ▸「Replicator を追加」を押す
  • ③ 既定で グリッド 3×3×3 / 間隔 200cm の複製が出ます

あとは「数」「間隔」「モード」などを変えると即座に反映されます。元のオブジェクト(複製元)はクローンとして表示されるため自動的に隠れます。

4. 画面の見かた(3つの要素)

「Replicator を追加」すると、内部的に3つの要素ができます。ふだん操作するのは Replicator 本体です。

要素役割操作
Replicator 本体パラメータを持つ親これを選んで N パネルで調整

パラメータはすべて Replicator 本体に保存され、ファイルを保存・再読込しても残ります。

6. 配置モード

クローンをどう並べるかを決めます。モードによって表示されるパラメータが変わります。

モード主なパラメータ
グリッド数 X / Y / Z、間隔(cm)X / Y / Z
リニア数、1ステップの移動(cm)X / Y / Z
放射数、半径(=間隔X / シンプルな円周)
円形数、半径(cm)、平面(XY/XZ/YZ)、角度(度)、軸を外向きに揃える
メッシュ参照メッシュ、配置先(頂点/辺/面/面上ランダム)、法線に揃える、揃える軸、変形後に追従
スプライン参照スプライン(カーブ)、数、接線に揃える、揃える軸

数の上限(セーフティ)

各モードの「数」は、セーフティ ON(既定)で最大 100、OFF で最大 1000 まで入れられます。

CAUTIONセーフティを外して大きな数にすると Blender が落ちる場合があります(特にグリッドは X×Y×Z で激増)。重い設定は少しずつ上げて様子を見てください。セーフティを ON に戻すと 100 超の数は 100 に丸められます。

メッシュモード(任意のメッシュ表面に配置)

任意のメッシュを「土台」にして、その 頂点 / 辺の中心 / 面の中心 / 面上ランダム にクローンを配置します。

  • 参照メッシュ: 土台にするメッシュ(複製元とは別物)。
  • 面上ランダム(散布): 面の上に面積重みで散らばせます(大きい面ほど多く乗る)。散布率(0〜1000%)とシードで密度・パターンを調整。散布率は前から増減するので、率を上げ下げしても既存の点は動きません(密度アニメ向き)。
  • 法線に揃える: クローンの軸を頂点/面の法線方向へ向けます。
  • 揃える軸: 法線へ向けるローカル軸を X/Y/Z/-X/-Y/-Z から選択。
  • 変形後に追従(評価メッシュ): 参照に付けたモディファイヤや変形を適用した後の形に乗せます(重いので必要な時だけ)。

スプラインモード(カーブに沿って配置)

カーブに沿って、クローンを弧長で等間隔に並べます。接線に揃えるで進行方向へ向けられます。閉じたカーブにも対応。

NOTE現状、複数のスプラインを持つカーブは最初の1本にのみ配置します。
REFERENCE

7. 複製元(参照オブジェクト)

複製の「元」になるオブジェクトです。メッシュ / ライト / 別の Replicator(入れ子)を使えます。複数登録もできます。

  • 追加: 対象を選択して「Replicator を追加」、または Replicator をアクティブにして「選択を複製元に追加」。
  • 一覧操作: パネルの複製元一覧から 選択 / 並べ替え(↑↓)/ 削除。

複数オブジェクトの分配 — ランダム / 規則的

複製元を2つ以上入れると、各クローンへどう割り当てるかを選べます。

  • ランダム: シードで各クローンにランダム割り当て。🔁(振り直し)で別の組み合わせを探せます。
  • 規則的(順番): 0,1,2,0,1,2… と順番に繰り返し割り当て。
  • 並べ替え = 割り当て順: 複製元一覧の ↑↓ で順番を変えると、複製の割り当て順もそれに追従します(規則的モードで効果が分かりやすい)。

複製元の位置はロックされます

「複製元を表示(編集用)」をオンにすると複製元が編集のため一時表示されますが、位置は常にロックされます(誤って動かしてクローンがズレるのを防ぐため)。

NOTEクローン全体を移動したいときは、「Replicator_Display を選択」ボタンで表示オブジェクトを選び、それを動かします(複製元ではなく Replicator_Display で位置を編集)。
CAUTION「複製元を表示」を ON のままにすると、複製元自体がレンダーに写ります。レンダー前は OFF にしてください。
REFERENCE

8. 段階トランスフォーム(複製ごとの変化)

位置・回転・スケールを複製ごとに少しずつ変化させます。Replicator 本体に内蔵された機能です。

  • 正規化(端→端 0→1)(既定 ON): 値は「最後のクローンでの変化量」。クローン数を増やしても破綻しません。
  • OFF にすると累積になります(数が多いほど変化が大きくなります)。
  • スケールは下限(0.001)で制限し、負やゼロになって反転するのを防いでいます。
REFERENCE

9. モジュレーター(ランダム / 段階 / 時間)

クローンに効果を与える「変調器」をリストで複数積めます。上から順に適用され、効果は加算合成されます。

  • 追加: 「+ Random」「+ Step」「+ Time」ボタン。置いた瞬間それらしく動く初期値入り。
  • 並べ替え / 削除 / ON・OFF: リスト右の ▲▼ / − ボタン、各行のチェック。
  • 共通: 位置(cm)/ 回転(度)/ スケール / 強度。
種類挙動
Randomクローンごとにランダム。シード + 🔁(振り直し)で好みの結果を探す
Stepクローン列に沿った勾配(0→1)。正規化 ON で数に依らず安定
Time時間で累積(キーフレーム不要)。速度で量を調整。回転に入れれば回り続ける

時間ランダム(Time): クローンごとに時間の進みをばらつかせ、位相をずらした有機的な動きにできます(シードは Random と共用)。

NOTE各モジュレーターの「フィールドで絞る」を ON にすると、そのモジュレーターだけが共有の範囲(フィールド)で局所化されます(次章)。
REFERENCE

10. 範囲(フィールド)— 効果を範囲で絞る

範囲の中にいるクローンだけ効果が効くように、クローンごとに 0〜1 の重みを掛けます(外は効果ゼロ)。共有フィールドが1つあり、「フィールドで絞る」を ON にした対象だけが局所化されます。

  • 追加: 「フィールド」行の右の力場アイコンでギズモ(エンプティ)が出ます。
  • 操作: ギズモを 移動(G)/ 回転(R)/ スケール(S)。動かすとクローンがリアルタイムに反応します。
  • 種類: 球 / 箱 / 方向(リニア)/ ノイズ。位置・向き・大きさはギズモ基準。
  • 反転: 内と外を入れ替え。

減衰をベジエ曲線で制御

「カーブで減衰を制御」を ON にすると、効きの強さ(芯=1 → 外=0)をベジエ曲線で自由に編集できます。点を足して、急に切る/滑らかに伸ばす/谷を作る、などが思いのままです。

NOTE球フィールドは等方なので回転しても同形です。向きを付けたいときは 箱 / 方向 / ノイズ を使ってください。
REFERENCE

11. ライトの複製

ライト(ポイント/スポット/エリア/サン)を複製元にすると、各クローン位置から実際に照らすライトが出ます(EEVEE 4.2+ / Cycles で確認)。

  • 配置モード・モジュレーター・フィールド・キーフレームもそのまま効くので、光のリング・光の道・面に散らした照明などが作れます。
  • 元のライト1つを編集すれば全クローンに反映(色・強さ・スポット角など)。
  • ライトは重いので、数を増やしすぎると遅くなります。
CAUTION「ジオメトリ化(Realize)」ではライトは出ません(形状を持たないため)。クローンライトは Realize OFF(既定)で使ってください。
REFERENCE

12. キーフレーム / アニメーション

ほぼ全パラメータをアニメーションできます。各パラメータの右の菱形ボタン(◆)で現在フレームにキーを打つ/消せます(標準の I キーや右クリックでも可)。

  • キーを打てる例: 間隔・半径(伸縮アニメ)、段階トランスフォーム、各モジュレーター、基準スケール、散布率(密度アニメ)、フィールド(半径/位置/角度)。
  • フィールド位置をキーすると、範囲が掃くように動き、通った所のクローンが順に反応します。
  • 再生・スクラブ・レンダーで毎フレーム再計算されます(キー不要の Time モジュレーターと併用も可)。
REFERENCE

13. 入れ子と「最上位を解除」

既存の Replicator を別の Replicator の複製元にすると入れ子になります。組み合わせるだけで複雑な表現を直感的に作れます。完全に非破壊で、内側を変えると外側も自動的に更新されます。

最上位を解除(本体だけ削除)

入れ子の最上位 Replicator だけを削除し、中身(子 Replicator や複製元)は構造を保ったまま独立させます。

  • ボタンは最上位の入れ子のとき(親が Replicator でない & 子に Replicator を含む)にだけ出ます。
  • ワールド位置は維持されます。構造を壊さず最上位だけ外したいときに使います。
REFERENCE

14. 実体化(Realize)/ 個別に展開

クローンは既定では「インスタンス」です。通常のモディファイヤをかけたり、実オブジェクトに書き出したいときは次の方法を使います。

  • クローン1個ずつ変形(全部同じ形に曲げる等): 複製元にモディファイヤを付ける(適用後の形が複製されます)。
  • 集合体全体を変形(全体を曲げる等): 「出力をジオメトリ化(Realize)」を ON → Replicator_Display にモディファイヤを追加。
  • 個別に展開(オブジェクト化): 現在の複製状態を編集可能な実オブジェクトとして個別に書き出します(非破壊・元はそのまま)。新しいコレクション「<名前>_Expanded」に並びます。
REFERENCE

15. シードについて

各ランダムのパターンを決める値です。分配シード / モジュレーターのシード / 散布シードがあり、🔁 で振り直して好みの結果を探せます。

  • シードの既定値は 123456 です。
  • マイナスは入力できません(0 以上)。古いファイルにマイナスが保存されていても、内部で正規化されるので問題なく開けます。
REFERENCE

16. 困ったとき / 既知の制限

  • レンダーで複製元が写る → 「複製元を表示」を OFF に。
  • クローンが消える / 反応が鈍い → パネルの「更新」ボタン、またはフレームを送ると再計算されます。
  • メッシュ/スプラインの変形が即反映されない → フレーム送りか「更新」で反映(変形アニメは再生で追従します)。
  • スプライン: 複数本のカーブは最初の1本のみ。
  • フィールド: 現状1個まで(複数ブレンドは今後)。
  • ツールチップ(説明文): 英語 UI でも一部は日本語のままです。

1. What is D_Replicator?

D_Replicator is a Blender add-on that clones an object into hundreds to thousands of copies and lets you build their layout and motion with real-time preview.

  • Non-destructive — the original object is untouched; changes apply instantly.
  • Lightweight — it draws a display-only mesh instead of creating real objects, so thousands stay fast.
  • Intuitive — good defaults the moment you add it; it builds up naturally.

Great for motion graphics, arrays/crowds, decoration, set dressing, and looping animation.

NOTEThis manual has two parts: Part 1 Quick Start (get going) and Part 2 Reference (every feature).
QUICK START

  • Pick the supplied D_Replicator-x.x.x.zip

6. Placement modes

Decide how clones are arranged. Parameters shown depend on the mode.

ModeMain parameters
GridCount X / Y / Z, Spacing (cm) X / Y / Z
LinearCount, step move (cm) X / Y / Z
RadialCount, radius (simple circle)
CircleCount, radius (cm), plane (XY/XZ/YZ), angle (deg), align outward
MeshReference mesh, placement (verts/edges/faces/surface), align to normal, axis, follow deformed
SplineReference spline (curve), count, align to tangent, axis

Count limit (safety)

Count is capped at 100 with Safety ON (default) and 1000 with it OFF.

CAUTIONVery high counts can crash Blender (Grid explodes as X×Y×Z). Raise heavy settings gradually. Turning Safety back ON rounds counts above 100 down to 100.

Mesh mode (place on any surface)

Use any mesh as a base and place clones on its vertices / edge centers / face centers / random surface scatter.

  • Reference mesh: the base (separate from the source being cloned).
  • Surface scatter: area-weighted random placement (bigger faces get more). Amount (0–1000%) and seed control density/pattern; points are added from the front so changing the rate keeps existing points fixed (good for density animation).
  • Align to normal / Align axis: aim a chosen local axis along the vertex/face normal.
  • Follow deformed: place on the shape after the reference's modifiers/deformation (heavier).

Spline mode

Place clones evenly by arc length along a curve. Align to tangent faces them along travel direction. Closed curves supported.

NOTECurrently only the first spline of a multi-spline curve is used.
REFERENCE

7. Sources

The objects clones are made from: mesh / light / another Replicator (nesting). Multiple allowed.

  • Add: select and "Add Replicator", or make the Replicator active and "Add Selected as Source".
  • List: select / reorder (↑↓) / remove from the panel list.

Distribution — Random / Iterate

When you register two or more sources, you can choose how each clone is assigned one.

  • Random: assign each clone a source by seed; 🔁 to reroll combinations.
  • Iterate (regular): assign 0,1,2,0,1,2… in order.
  • Reorder = assignment order: the ↑↓ in the source list change which source each clone gets (clearest in Iterate mode).

Source positions are locked

Turning on "Show Sources" reveals sources for editing, but their position is always locked (to avoid accidentally shifting the clones).

NOTETo move all clones, use the "Select Replicator_Display" button and move that object instead.
CAUTIONLeaving "Show Sources" ON renders the sources too. Turn it OFF before rendering.
REFERENCE

8. Step Transform

Shift position/rotation/scale progressively per clone. Built into the Replicator.

  • Normalize (end-to-end 0→1) (default ON): the value is the change at the last clone, so it stays sane as count grows.
  • OFF accumulates (dramatic with high counts). Scale is clamped (0.001) to avoid flips.
REFERENCE

9. Modulators (Random / Step / Time)

Stack multiple modulators; they apply top-to-bottom and add together.

  • Add: the "+ Random" / "+ Step" / "+ Time" buttons — each lands with sensible defaults so it animates immediately.
  • Reorder / remove / on-off: the ▲▼ / − buttons on the right of the list, and the checkbox on each row.
  • Common to all: position (cm) / rotation (deg) / scale / strength.
TypeBehaviour
RandomRandom per clone. Seed + 🔁 to explore
StepGradient along the clone sequence (0→1)
TimeAccumulates over time (no keyframes); Speed controls amount

Time Random scatters each clone's time progression for organic, out-of-phase motion (seed shared with Random).

NOTETurn on "Mask by Field" on a modulator to localize just that modulator with the shared field (next section).
REFERENCE

10. Field — localize effects

Multiplies each clone by a 0–1 weight so only clones inside the field are affected. One shared field; only targets with "Mask by Field" ON are localized.

  • Add: the force-field icon on the "Field" row creates a gizmo (empty).
  • Move (G) / Rotate (R) / Scale (S) the gizmo — clones react live.
  • Types: Sphere / Box / Linear (direction) / Noise. Invert swaps inside/outside.

Bezier falloff curve

Turn on "Falloff Curve" to edit the strength (core 1 → edge 0) with a free Bezier curve — add points for hard cuts, smooth ramps, or valleys.

NOTEA sphere field is isotropic (rotation does nothing). Use box / linear / noise for direction.
REFERENCE

11. Cloning lights

Use a light (point/spot/area/sun) as a source and each clone actually illuminates (verified on EEVEE 4.2+ / Cycles).

  • Placement, modulators, field and keyframes all apply — make light rings, light paths, scattered lighting.
  • Edit the single source light to update all clones (color, power, spot angle…).
  • Lights are heavy; too many will slow down.
CAUTIONLights do not appear when Realized (they have no geometry). Keep Realize OFF for clone lights.
REFERENCE

12. Keyframes / Animation

Almost every parameter animates. The diamond button (◆) next to a parameter inserts/removes a key at the current frame (standard I key / right-click also work).

  • Keyable: spacing/radius (grow/shrink), step transform, each modulator, base scale, scatter amount (density), field (radius/position/angle).
  • Keying the field position sweeps the area; clones react in turn as it passes.
  • Recomputed every frame on play/scrub/render (combine with the keyframe-free Time modulator).
REFERENCE

13. Nesting & "Dissolve Top"

Make a Replicator a source of another to nest them — complex results emerge just by combining. Fully non-destructive: editing the inner one updates the outer automatically.

Dissolve Top

Deletes only the outermost Replicator while keeping its contents (inner Replicators / sources) as an independent structure. World positions are preserved. The button appears only on the top of a nest.

REFERENCE

14. Realize / Expand to Objects

Clones are instances by default. Use the following when you need real modifiers or actual objects.

  • Deform each clone (bend identically, etc.): add a modifier to the source.
  • Deform the whole group: turn on "Realize Output", then add modifiers to Replicator_Display.
  • Expand to Objects: bake the current state into editable real objects (non-destructive; original kept) in a new "<name>_Expanded" collection.
REFERENCE

15. About seeds

Seeds drive the random patterns: distribution seed / modulator seed / scatter seed. Use 🔁 to reroll.

  • Default seed value is 123456.
  • Negatives cannot be entered (min 0). Old files that stored a negative still open fine (normalized internally).
REFERENCE

16. Troubleshooting / Known limits

  • Sources render → turn "Show Sources" OFF.
  • Clones vanish / sluggish → press "Refresh" or step a frame to recompute.
  • Mesh/spline edits not reflected instantly → step a frame or "Refresh" (deformation animation follows on playback).
  • Spline: only the first of multiple splines is used.
  • Field: one field for now (multi-field blending later).
  • Tooltips: some remain Japanese in the English UI.
